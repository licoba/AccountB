# AccountB：《账号册子》
--一个在本地记录、保存账号密码的App，增加了添加证件照的功能，保护个人隐私。

* 支持随机密码生成（待实现）
* 支持账号密码的备份和导入（待实现）
* 支持保存证件照
* 密码采用AES加密
